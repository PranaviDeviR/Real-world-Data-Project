# Healthcare Data Analysis Project

This project performs an Exploratory Data Analysis (EDA) on a healthcare dataset to uncover patterns in patient demographics, medical costs, and hospital operations.

## 🚀 Key Insights
* **Cost Drivers:** Analysis shows that billing amounts vary significantly by medical condition.
* **Length of Stay:** Calculated the treatment duration to identify operational trends.
* **Correlation Analysis:** Identified relationships between patient age, billing, and length of stay.
